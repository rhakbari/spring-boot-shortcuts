# InteliJ Spring-boot-shortcuts
```
Set your Inteli J keybinds to Visual studio in the settings
```

```
Alt + insert = to generate getter, constructor etc

Ctrl + N = new package class, file etc

Double Shift = Search Everywhere

F2= Navigate between code issues

Shift+F2 = Jump to the next or previous highlighted error.

Ctrl+Shift+Enter = Complete Current Statement

Insert any necessary trailing symbols and put the caret where you can start typing the next statement.

Ctrl+/ = Add/remove line or block comment

Ctrl+Shift+/ = Comment out a line or block of code.

Ctrl+B = Go To Declaration

Navigate to the initial declaration of the instantiated class, called method, or field.




```
